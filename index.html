WEB

The World Wide Web (WWW, W3) informs the world of information spaces over the Internet. Sometimes called wide web (web). This term is agreed with the Internet.

The web behaves like the Internet, just like e-mail. Since 1993, the web has occupied an absolute position in the Internet structure. Use the HTTP protocol, hypertext, HTML format, etc. on the Internet.
