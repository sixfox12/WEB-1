JavaScript

JavaScript is a scripting or programming language that allows you to apply something complex (such as periodic content updates, active maps, changing 2D / 3D graphics, video, etc.) to a web page.

JavaScript is the third layer in the layer cake called Standard Web Technology. More information about the other two layers (HTML and CSS) can be found elsewhere in the learning area.
